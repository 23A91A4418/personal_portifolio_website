# Gandham Rama Krishna Durga Prasanna

## Project Description

This is my personal portfolio website built to showcase my skills, projects, education, and experience in Data Science, Machine Learning, Deep Learning, and Large Language Models (LLMs). The website is fully responsive, includes smooth animations, parallax scrolling effects, and a professional modern UI to reflect my technical background and projects.

## Technology Stack Used

* React.js
* Vite
* Tailwind CSS
* JavaScript
* Framer Motion
* React Scroll Parallax
* HTML
* CSS
* Vercel (Deployment)

## Live Deployed URL

https://personal-portifolio-website-phi.vercel.app/

## Run Project Locally

### Clone Repository

git clone https://github.com/23A91A4418/personal_portifolio_website.git

### Install Dependencies

npm install

### Start Development Server

npm run dev

## Features

* Fully Responsive Design (Mobile + Tablet + Desktop)
* Hero Section with Resume Download
* About Me Section
* Skills Section
* Projects Showcase
* Contact Section
* Framer Motion Scroll Animations
* Parallax Scrolling Effect
* Accessibility Support using prefers-reduced-motion
* Professional UI Design
